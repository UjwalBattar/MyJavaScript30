  2. sort: (a, b) => a > b ? 1: -1
    if a is greater, returns 1 -> move to a higher index
    if a is smaller, returns -1 -> move to lower index
    if returns 0 -> remain in place in respect to each other but in order with rest of items

    });

    a.year > b.year ? 1 : -1;
    });
